## Getting Started

### Prerequisites

I build this entire project with the following setup:

- macOS Catalina version 10.15.7
- node v12.18.3
- npm v6.14.6
- yarn v1.22.5

### Development

1. create your `.env` file based on the `.env.example`

2. install all necessary dependencies

```bash
yarn install
```

3. start dev server

```bash
yarn start
```

### Production

1. install all necessary dependencies

```bash
yarn install
```

2. build application

```bash
yarn build
```

### Tests

run prettier check

```bash
yarn prettier-check
```

run all tests

```bash
yarn test
```
